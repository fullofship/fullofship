# FullOfShip

A WooCommerce shipping plugin.

## Description

FullOfShip is a custom shipping solution for WooCommerce.

## Requirements

- WordPress 5.0 or higher
- WooCommerce 5.0 or higher
- PHP 7.4 or higher

## Installation

1. Upload the plugin files to the `/wp-content/plugins/fullofship` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Configure the plugin settings in WooCommerce > Settings > Shipping
